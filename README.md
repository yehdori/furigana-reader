# Furigana Extension

A tiny Chrome extension I vibe-coded after getting an idea while studying Japanese.  
Highlight a Japanese sentence, request furigana, and it returns `<ruby><rt>` formatted text using the OpenAI API.

## Quick Start
1. Highlight a sentence
2. Trigger the extension
3. See furigana output

## Requirements
- OpenAI API key (usage may cost)
